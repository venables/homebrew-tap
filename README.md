# Homebrew Tap

Custom Homebrew formulas by [@venables](https://github.com/venables).

## Installation

```sh
brew tap venables/tap
```

## Formulas

| Formula                                              | Description                                                              |
| ---------------------------------------------------- | ------------------------------------------------------------------------ |
| [wt](https://github.com/venables/wt)                 | A simple git worktree manager                                            |
| [review-prs](https://github.com/venables/review-prs) | Pick open GitHub PRs and fan out parallel reviews into new terminal tabs |
| [anyagent](https://github.com/venables/anyagent)     | One non-interactive interface in front of any coding agent               |

### Install a formula

```sh
brew install venables/tap/wt
```
