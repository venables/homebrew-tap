# Homebrew Tap

Custom Homebrew formulas by [@venables](https://github.com/venables).

## Installation

```sh
brew tap venables/tap
```

## Formulas

| Formula | Description |
|---------|-------------|
| [wt](https://github.com/venables/wt) | A simple git worktree manager |

### Install a formula

```sh
brew install venables/tap/wt
```
